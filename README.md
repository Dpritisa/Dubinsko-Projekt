# Dubinsko-Projekt

# Virtualni asistent za referadu implementiran preko SVM i MLP

Za pokretanje projekt potrebno je:

preuzeti projekt : git clone https://github.com/Dpritisa/Dubinsko-Projekt

otvoriti powershell u folderu gdje je projekt

napraviti venv : python -m venv .venv

pokrenuti venv: .\.venv\Scripts\Activate.ps1

instalirati potrebne pakete: pip install -r requirements.txt 

istrenirati modele: python train.py 

  train_svm.py
                    
pokrenuti chatbot: python chat.py

tipkanjem /engine mlp ili /engine svm se mijenja model koji bot koristi

tipkanjem exit bot se prestaje izvoditi
